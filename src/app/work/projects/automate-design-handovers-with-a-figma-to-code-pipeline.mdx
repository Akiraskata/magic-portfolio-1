## Overview

本プロジェクトは、自身の学習成果や制作物を整理・発信することを目的として構築した、
パーソナルポートフォリオサイトです。

ブログ機能を通じて、プログラミングや Web 技術の学習内容を記録しながら、
プロジェクトページでは制作物や取り組みの過程を分かりやすくまとめています。
継続的な改善と機能追加を前提とした、長期運用可能な構成を意識しています。

---

## Key Features

- **MDX ベースのブログ・コンテンツ管理**
`.mdx` ファイルを用いて、記事やプロジェクト内容を簡単に追加・編集できる構成を採用しています。

- **ポートフォリオとブログの統合**
学習記録と制作物を一つのサイト内で管理し、成長の過程が分かる構成にしています。

- **シンプルで拡張性の高い設計**
将来的な機能追加（ユーザー管理や EC 機能など）を想定し、構造を整理しています。

- **画像・ビジュアルコンテンツ対応**
写真やスクリーンショットを活用し、視覚的にも理解しやすいページ構成を実装しています。

---

## Technologies Used

- **Next.js**
モダンなフロントエンド開発とページ構成のために使用。

- **TypeScript**
型安全性を確保し、保守性の高いコードを書くために採用。

- **MDX**
Markdown と React を組み合わせ、柔軟なコンテンツ管理を実現。

- **Java / Spring Boot（学習中・予定）**
将来的なバックエンド API 構築を想定し、現在学習を進めています。

---

## Challenges and Learnings

ファイルベースルーティングや MDX を用いたコンテンツ管理の仕組みを理解することが、
最初の課題でした。

試行錯誤を重ねる中で、ディレクトリ構成やプロジェクト設計が、
開発効率や保守性に大きく影響することを実感しました。
また、実装だけでなく「どのように整理して見せるか」を意識するようになりました。

---

## Outcome

本プロジェクトを通して、Web アプリケーションを一から構築し、
継続的に改善していく流れを実践的に学ぶことができました。

今後はバックエンド機能や EC 機能の追加にも取り組み、
フロントエンドとバックエンドを連携させた
フルスタック開発スキルの向上を目指しています。
