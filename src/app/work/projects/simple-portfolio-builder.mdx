## Overview

本プロジェクトは、自身の学習成果や制作物を一つの場所にまとめ、
継続的に記録・発信していくことを目的として構築した
パーソナルポートフォリオサイトです。

ブログやプロジェクト紹介を通して、
日々の学習内容や試行錯誤の過程を残しながら、
将来的な機能拡張にも対応できるよう、
シンプルで柔軟な構成を意識して設計・実装しています。

---

## Key Features

* 学習記録・制作物を一元的に管理できる構成
* ブログ・プロジェクトを分離した情報設計
* コンテンツ追加を前提とした拡張しやすい構造
* 継続的な更新を想定したシンプルな運用設計

---

## Tech Stack

* Backend：Java / Spring Boot
* API：REST API（学習・実装中）
* Frontend：Next.js / React
* Styling：CSS / Component-based design

---

## Purpose

学習内容や制作物が分散しやすいという課題を感じ、
「学び・制作・発信」を一つの流れとして整理できる
ポートフォリオを構築したいと考え、本プロジェクトを開始しました。

単なる成果物の展示ではなく、
学習の過程や思考の変化も含めて記録することで、
自身の成長を振り返ることを目的としています。

---

## Future Improvements

* 管理画面の実装による記事更新フローの改善
* データ構造の整理と再設計
* 認証機能を用いた運用機能の拡張
* バックエンド機能の段階的な強化
